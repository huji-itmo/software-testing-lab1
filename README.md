### Задание 1: Функция arctg(x)

**Реализация:** `arctan/src/main/kotlin/org/example/app/ArctanKt.kt`
- Функция `arctg(x)` — разложение в степенной ряд Тейлора: `arctg(x) = x - x³/3 + x⁵/5 - x⁷/7 + ...`
- Оптимизация для |x| > 1: `arctg(x) = π/2 - arctg(1/x)`
- Функция `arctgPrecise(x)` — с заданной точностью

**Тесты:** `arctan/src/test/kotlin/org/example/app/ArctanTest.kt`
- Параметризованные тесты (JUnit 5)
- Покрытие: граничные значения (x = -1, 0, 1), дробные значения, большие значения |x| > 1, симметричность
- Использование `kotlin.math.atan()` для эталонных значений

**Результат:** 9 тестов — все проходят
